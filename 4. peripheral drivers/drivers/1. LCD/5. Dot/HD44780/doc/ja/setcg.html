lcd_setcg

ユーザ文字をLCDCに登録します。

void lcd_setcg (
    uint8_t chr,       /* 先頭の文字コード */
    uint8_t n,         /* 文字数 */
    const uint8_t* pat /* 文字パターンへのポインタ */
);

引数

chr
登録するユーザ文字の先頭のコード(0〜7)を指定します。
n
登録するユーザ文字の数。
pat
文字パターン(n×8バイトの配列)へのポインタ。

解説

ユーザー文字のパターンを登録します。文字パターンはLCDCのCGRAMに書き込まれ、表示中のユーザ文字には即時反映さえます。デュアル・コントローラの場合は、両方のLCDCに同じパターンが書き込まれます。

対応情報

_USE_CGRAMが1のとき使用可能です。
