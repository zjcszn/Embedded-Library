State Machines

Purpose and Scope

This document (Unique Identifier: DOC-QP-SDS) describes the detailed design of the QP Real-Time Embedded Framework (RTEF). This software design description is designed to be most helpful to the QP Application developers, who are also the primary users of the QP Framework.

Remarks
This document is part of the QP Certification Kit, which has been specifically designed to aid companies in safety certification of their software based on the QP Framework treated as commercial off-the-shelf (COTS) software.
